# ML Models from Scratch

Machine learning algorithms implemented from scratch using NumPy and Pandas — no sklearn for core models. Every algorithm is built manually to understand the underlying math and logic, covering the full pipeline from raw data to evaluation.

## What this repo covers

- **Supervised Learning** — KNN, Linear Regression, Logistic Regression, Naive Bayes, Decision Trees, Random Forests, Ensemble methods
- **Data Preparation** — missing value handling, feature engineering, feature scaling, data cleaning
- **Model Evaluation** — cross-validation, hyperparameter tuning, regularization, evaluation metrics
- **Imbalanced Data** — techniques for handling imbalanced datasets
- **Unsupervised Learning** — clustering and unsupervised methods
- **Reinforcement Learning** — intro-level RL concepts and implementations

## Structure

Each folder contains a standalone project with implementation, experiments, and tests.
